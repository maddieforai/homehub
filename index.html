💡 Top Tips ▾
  • Help us come back to a clean, refreshing home. Familiarise yourself with where things are kept so you can put things back where they belong.
  • Minimise things on surfaces unless they are meant to be there — e.g. photos, flowers, lights, candles, tissue boxes.
  • Small things make the difference — how display items are placed, the direction things face (e.g. cushions), wardrobe organisation.
  • Communication is key. Feel free to check if things need to be thrown out. It's ok to put things away, but try to remember where — always put the same things back in the same place.
  • Tidy up as soon as possible, so the place is always neat and clear.
